Hugo-smorg is an open source theme for Hugo, the fast and flexible static site engine written in Go. The theme incorporates Bulma, a responsive CSS framework and Bulmaswatch theme customizations for Bulma.

Hugo-smorgは、Goで書かれた高速かつ柔軟な静的サイトエンジンであるHugoのオープンソースのテーマです。このテーマには、BulmaのレスポンシブなCSSフレームワークとBulmaswatchのテーマカスタマイズが組み込まれています。
